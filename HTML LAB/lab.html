Differentiation Tag

Tag ความเหมือน ความแตกต่าง
hr & br เป็นการแบ่งหรือแยกเนื้อหาข้อมูลออกเหมือนกัน hr แบ่งโดยใช้เส้น br แบ่งโดยการเพิ่มเว้นหนึ่งบรรทัด
ul & ol เป็นการเรียงออกมาในรูปแบบลิสเหมือนกัน ul คือ unordered list ซึ่งจะเรียงในแบบของจุด
ol คือ ordered list ซึ่งจะเรียงโดยใช้ตัวเล็ก
select & checkbox & radio button เป็นแบบฟอร์มของตัวเลือกเหมือนกัน select เป็นรูปแบบของ dropdown lists
checkbox เป็นรูปแบบของการกดเลือกในกล่องสี่เหลี่ยมซึ่งสามารถเลือกได้มากกว่าหนึ่งช่อง
radio button เป็นรูปแบบของการกดเลือกในกล่องสี่เหลี่ยมได้เพียงกล่องเดียว
div & span เป็น Element เหมือนกัน div เป็น block element จะถูกจัดเรียงกันลงมา
span เป็น inline element จะถูกจัดเรียงติดกันโดยอยุ่ในบรรทัดเดียวกัน
script & style เป็น internal tag ที่ใช้ในไฟล์ html เหมือนกัน script เป็นแทคสำหรับการเขียน javascript
style เป็นแทคสำหรับการตกแต่งกับ css

Input Type

Input Type ใช้อย่างไร
input type="button" ใช้กำหนด value ให้เป็นปุ่มเพื่อกดดำเนินการต่อ
input type="checkbox" ใช้กำหนด value ให้เป็นที่เลือกข้อมูลที่สามารถเลือกได้มากกว่าหนึ่ง
input type="color" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลของค่าสี
input type="date" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลวันที่
input type="datetime-local" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลเวลา ณ ปัจจุบันตามสถานที่นั้นๆ
input type="email" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลอีเมล์
input type="file" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลแบบอัพโหลดไฟล์
input type="hidden" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลแบบไม่แสดงบนหน้าจอและเพื่อนำไปใช้
input type="image" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลแบบอัพโหลดรูปภาพ
input type="month" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลเดือน
input type="number" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลชนิดตัวเลข
input type="password" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลรหัสผ่าน
input type="radio" ใช้กำหนด value ให้เป็นที่เลือกข้อมูลที่สามารถเลือกได้แค่หนึ่งอย่างเท่านั้น
input type="range" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลที่สามารถกำหนดช่วงของข้อมูลนั้นได้
input type="reset" ใช้กำหนด value ให้เป็นปุ่มเพื่อเร่ิมการใส่ข้อมูลใหม่อีกครั้ง
input type="search" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลเพื่อการค้นหา
input type="submit" ใช้กำหนด value ให้เป็นปุ่มเพื่อกดส่งข้อมูล
input type="tel" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลหมายเลขเบอร์โทรศัพท์
input type="text" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลตัวหนังสือ
input type="time" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลเวลา
input type="url" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลที่อยู่ ​URL
input type="week" ใช้กำหนด value ให้เป็นที่ใส่ข้อมูลสัปดาห์
